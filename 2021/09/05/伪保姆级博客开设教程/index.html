创建仓库

  • 登录github，点击右上角头像 旁边 的加号

  • 点击 new repository

  • 在 repository name 中填入 username.github.io，其中username为你的用户名。

    比如你叫的用户名为 shenben，就填入 shenben.github.io

    然后创建。

    注意：

    • 注册的邮箱记得验证
    • 由于奇怪的原因，部分仓库创建后需要等待一段时间才会显示。

  • 创建完成后不用管跳出来的东西。
